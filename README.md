# ERP Nexus API

API REST para sistemas ERP desenvolvidos com Delphi, Horse e Firebird.

## Status

🚧 Projeto em desenvolvimento.

## Objetivo

Construir uma API REST demonstrando a modernização de sistemas ERP desenvolvidos em Delphi através de uma arquitetura em camadas, integração REST, autenticação, testes automatizados e integração com banco de dados.

## Tecnologias

- Delphi 10.4 Sydney
- Horse
- FireDAC
- Firebird
- Git

## Funcionalidades Planejadas

- API REST
- Gestão de clientes
- Gestão de produtos
- Estoque
- Pedidos
- Autenticação
- Suporte a multiempresa
- Testes automatizados
- Documentação da API
